Real-time communication application using **WebRTC** for peer-to-peer media with signaling via backend services—built to demonstrate realtime engineering and networking concepts.

> 🔗 **Backend Repo:** `muhamadzainalam-dev/VideoCallingApp-Be`  
> `https://github.com/muhamadzainalam-dev/VideoCallingApp-Be`

---

## 💼 Skills & Tech Used

- **UI:** React, React DOM
- **Build Tooling:** Vite
- **Routing:** React Router, React Router DOM
- **Realtime Signaling:** Socket.IO Client (`socket.io-client`)
- **Media/Playback:** React Player
- **Code Quality:** ESLint + React Hooks/Refresh plugins

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/VideoCallingApp-Fe.git
cd VideoCallingApp-Fe
npm install
npm run dev
```

Open:
```text
http://localhost:5173
```
